Prueba todas las funcionalidades del Simulador Cuántico

🎯 API REST Simple

Simulación cuántica básica con respuesta inmediata. Ideal para pruebas rápidas y desarrollo.

Haz clic en el botón para ejecutar una simulación simple.

🔌 WebSocket Tiempo Real

Simulación con progreso en tiempo real usando WebSockets. Observa el progreso paso a paso.

Haz clic en el botón para iniciar una simulación WebSocket.

🧪 API Mock

Simulación mock para desarrollo frontend. Respuestas predefinidas sin cálculos reales.

Haz clic en el botón para ejecutar una simulación mock.

📊 Comparación de APIs

Ejecuta la misma simulación con diferentes APIs para comparar resultados y tiempos.

Haz clic en el botón para comparar diferentes APIs.

🔬 Análisis de Interacción

Analiza la interacción entre dos moléculas diferentes usando métodos cuánticos.

Selecciona dos moléculas y haz clic en el botón para analizar su interacción.

📚 Información de Moléculas

Obtén información detallada sobre las moléculas disponibles en el simulador.

Haz clic en el botón para obtener información de las moléculas.
